System Architecture

Data Flow

  • Dataset from Roboflow Universe
  • Train/validation/test split (70/20/10)
  • Augmentation: rotation, brightness, flip
  • YOLOv10 trained → best.pt

Inference Pipeline

  • YOLOv10 → bounding box + crop
  • Image enhancement (CLAHE, sharpening)
  • Gemini 2.5 Flash → OCR
  • Blacklist match → Gmail SMTP alert
